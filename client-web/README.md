# client-web

## Project setup
```
npm install
```

You should also re-run this after `git pull`.

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints files
```
npm run lint
```

### Lints and fixes files
```
npm run lint-fix
```
